# NemesisNet WordPress Theme

A custom WordPress theme built for the NemesisNet brand, featuring modern design aesthetics and comprehensive functionality.

## Theme Files

- style.css - Main theme stylesheet with theme metadata
- unctions.php - Theme functions and WordPress hooks
- header.php - Header template
- ooter.php - Footer template
- index.php - Main template file
- screenshot.png - Theme screenshot for WordPress admin

## Installation

1. Copy this entire directory to your WordPress themes folder: wp-content/themes/
2. Navigate to **Appearance > Themes** in your WordPress admin panel
3. Activate the **NemesisNet** theme

## Features

- Custom theme settings and configurations
- Modern, responsive design
- NemesisNet brand integration
- Optimized for performance

## Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher
